# AI-Based Custom Recipe Tutorial Generator

A web application that generates personalized cooking recipes based on the ingredients you have available. This project uses OpenAI's API and Flask to provide step-by-step recipe tutorials along with fun facts about each dish. Built with Python, HTML, CSS, and the Flask framework, this app offers an interactive and user-friendly experience.

---

## Features

- **AI-Powered Recipe Suggestions**: Enter the ingredients you have, and the AI will generate a recipe.
- **Step-by-Step Instructions**: Clear and easy-to-follow cooking steps.
- **Fun Facts**: Each recipe comes with an interesting fact about the dish.
- **Copy to Clipboard**: Quickly save your recipe.
- **Responsive Web Interface**: Built with HTML, CSS, and Bootstrap for a clean design.
---

## Technologies Used

- **Python 3.x**
- **Flask** – Web framework
- **OpenAI API** – AI recipe generation
- **HTML & CSS** – Frontend design
- **JavaScript** – Responsive layout
